# StudyPlanner

- Visual studio code with python virtual environment
- Run .venv\Scripts\Activate.ps1 to activate venv

To run back-end \
$env:FLASK_APP="app.py"  \
$env:FLASK_ENV="development"\
py -m flask run 

To run front-end \
npm start 

Languages used:
- Python 3+
- Node.js

Frameworks:
- Flask
- React.js

DB:
- MongoDB + mongoengine
